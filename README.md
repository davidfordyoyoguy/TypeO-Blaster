TypeO Blaster is a typing app that trains speed and accuracy. 
There are various modes to choose from and additional options that you can add to any mode to focus on speed, accuracy and penalizing backspace spamming. 
There is a functioning keyboard visualizer that guides which fingers to use on each key and the current key(s) to hit next in the prompt.
On the stats page, there is an option to clear the stats.
I will add features and fixes over time.
